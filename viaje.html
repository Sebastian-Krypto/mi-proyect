DNS

DNS traduce nombres de dominio a direcciones IP.

Direcciones IP

Las IP identifican dispositivos en una red.

Proceso

  1. Usuario escribe una URL.
  2. DNS busca la IP.
  3. Conexión con el servidor.
  4. Respuesta del servidor.
